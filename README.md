# Ownership Lifecycle Declaration Engine (Prototype Brain)

A tiny, runnable **state declaration engine** that models a confidential business ownership lifecycle.
It turns private exit uncertainty into structured early transition action — **without forcing premature public disclosure**.

## Core Thesis
Communities don’t hollow out when businesses fail.  
They hollow out when **ownership transitions fail**.

Succession is treated as an end-of-life crisis instead of a lifecycle state.  
That misclassification is the invisible failure.

## What this is (and is not)
**This is:**
- A lifecycle-state engine for owners to **declare** where they are
- A rules-based module unlock system (state → tasks/resources)
- A lightweight progress indicator (e.g., “succession readiness: 30%”)
- Optional visibility toggles (private by default; owner-controlled public signal)

**This is not:**
- A marketplace
- A chamber dashboard
- A valuation tool
- Growth coaching
- A matchmaking platform

## Lifecycle states
- GROWTH
- STABLE
- SUCCESSION_DEVELOPING
- EXIT_CURIOUS
- EMERGENCY_CONTINUITY_ONLY

## Mechanism (the hinge)
1) Owner declares a lifecycle state (private by default)  
2) State unlocks tailored modules instantly  
3) Owner completes small steps → progress becomes visible  
4) Owner may optionally make the state public later (signal without details)
